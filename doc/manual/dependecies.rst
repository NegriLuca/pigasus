Dependecies
***********

Pigasus is a Generic Python Library for IsoGeometric Analysis simulations.

The following Python modules are needed:

* numpy

* scipy

* matplotlib

You will also need a Fortran compiler [#f1]_ and the fortran wrapper *f2py*.
